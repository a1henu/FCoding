# FCoding

Your Feishu bot should run Codex, reply in chat, and handle card callbacks.

We already wrote the setup path down.

Give it to an agent. Let the agent do the configuration work.

## Installation

### For Humans

Copy and paste this prompt to your AI coding agent:

```text
Install and configure FCoding by following the instructions here:
https://raw.githubusercontent.com/a1henu/FCoding/refs/heads/main/docs/agent-installation.md
```

或把下面这段中文 prompt 直接发给你的 AI coding agent：

```text
请按照这里的说明安装并配置 FCoding：
https://raw.githubusercontent.com/a1henu/FCoding/refs/heads/main/docs/agent-installation.md
```

### For Agents

Read and follow:

- https://raw.githubusercontent.com/a1henu/FCoding/refs/heads/main/docs/agent-installation.md

Use the repository source of truth. If docs and code disagree, say so explicitly before changing behavior.

Or read the installation guide in the repo, but it is faster to let an agent handle the setup.
